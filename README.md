# Snake Game (C# Console)

This is a simple Snake game built as a console application in C#.

---

## Features

* Move the snake using arrow keys
* Snake grows when eating food
* Score system
* Collision detection:

* Walls
* Snake itself
* Restart functionality

---

## Technologies

* C#
* .NET (Console Application)
* Developed in Rider

---

## How to Run

1. Clone the repository:

   ```bash
   https://github.com/EddieJohanssonn3000/snake-game.git
   ```

2. Open the project in Rider

3. Run the program:

4. Play
   
---

## Controls

* ⬆️ Up Arrow – Move up
* ⬇️ Down Arrow – Move down
* ⬅️ Left Arrow – Move left
* ➡️ Right Arrow – Move right
* **R** – Restart after Game Over
